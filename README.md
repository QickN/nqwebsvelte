# nickquam.com

SvelteKit source for [nickquam.com](https://nickquam.com), the personal product engineering site
for Nick Quam.

## Purpose

This site is the public front door for Nick's current work:

- [Checkout](https://checkoutrankings.com) - native iPhone grocery rankings app
- [Astretto](https://astretto.com) - focused consumer software studio

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run check
npm run build
```

## Deployment

The live domain is served by the connected Cloudflare Pages deployment for this repository.
Pushes to `main` publish the site.
